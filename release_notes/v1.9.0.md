# Daily Rhythm Companion v1.9.0 release notes

## Theme

v1.9.0 establishes Daily Rhythm Companion as a public-safe smartphone Web demo app for AI Character Framework v4.0.0 capability verification.

## Highlights

- Verified FW4.0.0 LLM/text-chat through the actual DRC backend API and smartphone Web UI.
- Routed a live framework text-chat reply through the DRC post-advice chat adapter.
- Recorded public-safe smartphone Web evidence for the STT / voice input boundary.
- Recorded public-safe smartphone Web evidence for the TTS / voice output boundary.
- Recorded public-safe smartphone Web evidence for the Live2D / VTS motion boundary.
- Added explicit opt-in gates for live framework/provider execution.
- Preserved public-safe evidence policies: no API keys, raw LAN IPs, prompt bodies, response bodies, raw screenshots, audio artifacts, motion payloads, or provider payloads are committed.
- Verified the fixed release zip without rebuilding it during final checks.

## Scope notes

v1.9.0 is a public demo / verification release. It is not an app-store consumer release.

The LLM/text-chat path includes one explicit live framework/provider smoke path under local operator opt-in gates. STT, TTS, and Live2D/VTS are verified as guarded smartphone Web boundaries, not as production runtime execution paths.
