# Daily Rhythm Companion v2.0.1

Status: PREPARED / NOT_RELEASED
Release tag: planned `DRC_v2.0.1`
Released baseline: v2.0.0

## Summary

v2.0.1 is a maintenance and regression-hardening patch for the released v2.0.0 capability set. It does not add a large new user-facing subsystem and does not replace or rewrite the v2.0.0 release.

## Maintenance changes

```text
- Align backend runtime/OpenAPI metadata to 2.0.1 and Flutter package metadata to 2.0.1+2.
- Add a normal credential-free backend pytest suite for core API, mock advice, and DailyRecord behavior.
- Add regression coverage for AI Character Framework success/fallback metadata and DRC-owned voice artifact safety.
- Bound process-local post-advice chat sessions with idle TTL and LRU capacity limits.
- Bound DRC-owned TTS staging/public artifacts with TTL, count limits, and lazy cleanup.
- Make Web CORS origins configurable while preserving the released wildcard local-demo default.
- Clarify mock, wearable_stub, deprecated fitbit_stub, and legacy Fitbit migration/reference states.
- Add one aggregate v2.0.x maintenance readiness command.
- Add the M-9 committed-source, one-time fixed-ZIP, same-artifact verification, and SHA-256 record gates.
```

## Compatibility

```text
- Existing v2.0.0 API routes and response models remain compatible.
- Existing Google Health, configured LLM, TTS, image, smartphone Web, DailyRecord, history, and report behavior remain the accepted baseline.
- Mock-safe defaults remain credential-free.
- Existing opaque voice artifact URL and 404 behavior remain compatible.
- WEB_CORS_ORIGINS defaults to * for local-demo compatibility.
- Existing Fitbit routes and response fields remain available; real Fitbit completion stays deferred to v2.1.0.
```

## Not included

```text
- real microphone capture or STT execution
- realtime voice orchestration, cancellation, interruption, or barge-in
- real Live2D or VTube Studio adapter execution
- user accounts, cloud synchronization, or production multi-user hosting
- App Store or Google Play publication
- configured real Fitbit completion or operator acceptance
```
