# Daily Rhythm Companion v1.1.0

Daily Rhythm Companion v1.1.0 is a post-v1.0 cleanup and public repository readiness release.

This release does not add a large new app feature. Instead, it prepares the project for safer public repository publication after the v1.0.0 demo-app milestone.

## Release package

```text
release\DailyRhythmCompanion_20260521_110149.zip
```

## Highlights

- Added a v1.1.0 public repository publication plan.
- Updated the roadmap baseline from v1.0.0 release completion to v1.1.0+ development.
- Added docs inventory policy for public repository cleanup.
- Added scripts inventory policy for public repository cleanup.
- Added safe cleanup structure for future docs/scripts archiving.
- Added `docs/archive/README.md`.
- Added `scripts/archive/README.md`.
- Updated `scripts/README.md` for v1.1.0 public repo readiness while preserving v1.0.0 compatibility markers.
- Added v1.1.0 public repo readiness gate.
- Preserved v1.0.0 release/final/compatibility verification behavior.

## Known limitations

v1.1.0 is a public repository readiness and cleanup-policy release.

It intentionally does not claim:

```text
- production hosted consumer-service readiness
- App Store / Google Play distribution
- production-grade Google Health onboarding
- mandatory microphone support
- mandatory TTS provider support
- mandatory Live2D/VTS runtime
- always-on realtime assistant behavior
```
